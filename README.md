# 📸 SHREPHO – Gesture-Controlled Photo Booth

SHREPHO is an interactive, touchless photo booth built using **HTML, CSS, and JavaScript** with **MediaPipe Hands** for real-time hand gesture recognition. Users can frame photos, capture images, solve a fun puzzle, and generate customizable photo strips—all without touching the screen.

## ✨ Features

- 🤚 Touchless gesture-based controls
- 📷 Draw a photo frame using hand pinches
- ⏳ Countdown before capturing photos
- 🎞️ Black & white vintage film effect
- 🧩 Interactive 3×3 puzzle after every capture
- 🖼️ Generate a 3-photo strip
- 🎨 Multiple photo strip themes
  - Retro
  - White
  - Wooden
  - Noir
- 💾 Download the final photo strip as PNG
- 📱 Fully responsive interface
- 🔒 Runs completely in the browser (no server required)

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API
- MediaPipe Hands
- Camera API (getUserMedia)

## 🚀 How It Works

1. Allow camera access.
2. Pinch with both hands to create a photo frame.
3. Hold steady to lock the frame.
4. Pinch the capture button or make a fist to take a photo.
5. Solve the generated puzzle.
6. Repeat until all three photos are captured.
7. Choose a photo strip theme.
8. Download your final photo strip.
